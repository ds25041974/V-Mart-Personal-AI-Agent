1 Quick Setup Guide - V-Mart Personal AI Agent

1.1 ‚ö° Get Your Gemini API Key (Required for Demo Mode)

1.1.1 Step 1: Get Gemini API Key

  1. Go to Google AI Studio
  2. Click ‚ÄúGet API Key‚Äù or ‚ÄúCreate API Key‚Äù
  3. Select or create a Google Cloud project
  4. Click ‚ÄúCreate API Key in new project‚Äù (or use existing project)
  5. Copy the API key (starts with AIza...)

1.1.2 Step 2: Update .env File

Open the .env file and replace:

GEMINI_API_KEY=your_gemini_api_key_here

With your actual API key:

GEMINI_API_KEY=AIzaSy...your_actual_key...

1.1.3 Step 3: Restart the Server

# Stop the current server (Ctrl+C)
python3 main.py

1.2 üîê Full OAuth Setup (For Production - All V-Mart/Limeroad Users)

If you want to enable Google OAuth login for all users from your domains:

1.2.1 Step 1: Google Cloud Console Setup

  1. Go to Google Cloud Console
  2. Create a new project or select existing
  3. Enable Google+ API and Google Identity services

1.2.2 Step 2: Create OAuth Credentials

  1. Navigate to APIs & Services > Credentials
  2. Click ‚ÄúCreate Credentials‚Äù > ‚ÄúOAuth 2.0 Client ID‚Äù
  3. Configure consent screen:
    • User Type: Internal (for workspace users) or External
    • App name: V-Mart Personal AI Agent
    • Authorized domains: Add vmart.co.in, vmartretail.com, limeroad.com
  4. Create OAuth Client ID:
    • Application type: Web application
    • Authorized redirect URIs: http://localhost:8000/auth/authorize
  5. Copy Client ID and Client Secret

1.2.3 Step 3: Update .env File

GOOGLE_CLIENT_ID=your_actual_client_id.apps.googleusercontent.com
GOOGLE_CLIENT_SECRET=your_actual_client_secret

1.2.4 Step 4: Domain Restrictions

The app is already configured to only allow users from: - vmart.co.in - vmartretail.com - limeroad.com

Users from other domains will be rejected automatically.

1.7 ‚ùì Troubleshooting

1.7.1 ‚ÄúAPI key not valid‚Äù Error

  • Make sure you copied the entire API key (starts with AIza)
  • Check for extra spaces or quotes in .env file
  • Verify the key is active in Google AI Studio

1.7.2 OAuth Login Not Working

  • For now, use Demo Mode to test
  • Full OAuth setup requires Google Cloud Console configuration
  • See ‚ÄúFull OAuth Setup‚Äù section above

1.7.3 Server Won‚Äôt Start

  • Verify Python version: python3 --version (should be 3.8+)
  • Check port 8000 is not in use: lsof -i :8000
